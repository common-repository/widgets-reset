A guide to Widgrets Reset

Widgets Reset is a plugin that resets all of your widgets. Things go back to the factory settings, so to speak. This is useful when changing themes or if your widgets get stuck somewhere and you don't know where they're at.

Please note that this will wipe everything clean. Any widgets you have saved will be completely removed. So, if you have important information in a particular widget, you better save it before hitting the submit button.

How to install the plugin

  1. Uzip the widgets-reset.zip folder.
  2. Upload the widgets-reset folder to your /wp-content/plugins directory.
  3. In your WordPress dashboard, head over to the Plugins section.
  4. Activate Widgets Reset.

How to use the plugin

Once you've installed and activated the plugin, a new menu item will appear under Appearance in your WordPress dashboard. It will be labeled Widgets Reset. Click on it.

On the new page, all you must do is click the button that reads Reset All Widgets. This will remove all widgets from active status on your site.

After this, you can disable the plugin unless you want to keep it around to use later.
